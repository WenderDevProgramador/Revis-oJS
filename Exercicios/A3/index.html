Desafio:

Escreva um programa que recebe 3 argumentos, uma frase, uma palavra e um array de indices. A função deve retornar a frase com a palavra inserida em cada uma das posições especificadas pelo array de indices.

- Função não deve funcionar em indices que não estejam ao alcance da frase

- Funçaõ deve retornar a mesma frase caso o array de indices esteja vazio.

Exemplos são:

- Entrada: capaz ultilizar as cápsulas emergencia, de, 6,27
- Saída: capaz de ultilizar as cápsulas de emergencia
