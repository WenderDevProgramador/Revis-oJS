Desafio:

Escreva um programa que use uma classe que sejá capaz de lidar com uma equação de segundo grau.Os objetos podem ser instanciados com valores padrão a=0, b=0, c=0. Ela ela deve ter um método que retorna suas raizes na forma de um array vvazio, de um ou de dois elementos. O método também deve conter os passos para a resolução da equação. Se a,b,c forém igual a zero ela deve retorna um erro.

Exemplos são:

Entrada: 0,2,6
Saída:
1. 2x + 6 = 0
2. 2x = -6
3. x = -6/2
4. x = -3
Raiz: [-3]
