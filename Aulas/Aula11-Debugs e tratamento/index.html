Debug e Tratamento de Erro

Oque é bug e debug?

bug: Problema que ocorre no codigo muitas vezes ocasionado por erro do programador.

debug:O ato de resolver os bugs encontrados no código.

Strict mode: Usamos para diminuir um pouco da liberdade do JS e deixando mais rigoroso para evitar erros

console.log():Usamos em pontos estrategicos do codigo para vizualizar dados que podem está confusos

ou identificar aonde o codigo apresenta o erro e para de rodar.

Método de debug "debugger"Faz o código parar no momento do erro.
