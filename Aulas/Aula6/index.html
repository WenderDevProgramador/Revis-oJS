Protocolos da web

* HTTP: Serve para navegar, solicitar arquivos e imagens no servidor

* SMTP: Protocolo para envio de emails

* TCP: Protocolo para transferência de dados como dowloads

DOM

O DOM (Document Object Model) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura do documento como uma árvore de nós, onde cada nó é um objeto que pode ser manipulado com JavaScript.

O DOM permite que os desenvolvedores acessem e modifiquem o conteúdo, a estrutura e o estilo de um documento HTML ou XML dinamicamente. Isso significa que você pode alterar elementos, adicionar novos elementos, remover elementos existentes e alterar atributos de elementos em tempo real.
